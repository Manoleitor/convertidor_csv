# Convertidor de CSV a Hoja de Asistencia

Este repositorio contiene herramientas para convertir archivos CSV en hojas de asistencia (hojas de pasar lista).

## Propósito
Transformar datos CSV en documentos de seguimiento de asistencia formateados para la gestión del aula.

## Características
- Convertir archivos CSV a formato de hoja de asistencia
- Proceso de conversión fácil de usar
- Formato de salida personalizable
